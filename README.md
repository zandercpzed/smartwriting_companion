## Português

SmartWrite Companion é um plugin para Obsidian que fornece análises em tempo real de escrita para melhorar sua produtividade e acompanhar seu progresso.

### Funcionalidades Principais

- **Análise em Tempo Real**: Conta palavras, parágrafos, frases e sílabas instantaneamente enquanto você escreve
- **Métricas de Leitura**: Calcula tempo de leitura estimado e velocidade de leitura (WPM)
- **Rastreamento de Sessão**: Monitora tempo de escrita da sessão atual e progresso diário
- **Metas Diárias**: Configure e acompanhe metas de palavras por dia
- **Interface Modular**: UI em acordeão com seções expansíveis para métricas, sessão e configurações
- **Otimização de Performance**: Sistema de cache inteligente para atualizações instantâneas
- **Configurações Personalizáveis**: Painel de configurações para visibilidade de módulos e metas

### Instalação

1. Baixe os arquivos do plugin
2. Copie `main.js`, `styles.css` e `manifest.json` para `.obsidian/plugins/smartwrite-companion/`
3. Reinicie o Obsidian e ative o plugin nas configurações

### Uso

Após ativar o plugin, uma nova visualização lateral "SmartWrite Companion" estará disponível. Abra-a para ver as métricas em tempo real enquanto edita seus documentos.

### Desenvolvimento

Para desenvolvimento local:
```bash
npm install
npm run build
```

## English

SmartWrite Companion is an Obsidian plugin that provides real-time writing analytics to enhance your productivity and track your progress.

### Main Features

- **Real-Time Analysis**: Instantly counts words, paragraphs, sentences, and syllables as you write
- **Reading Metrics**: Calculates estimated reading time and reading speed (WPM)
- **Session Tracking**: Monitors current session writing time and daily progress
- **Daily Goals**: Set and track daily word count goals
- **Modular Interface**: Accordion UI with expandable sections for metrics, session, and settings
- **Performance Optimization**: Intelligent caching system for instant updates
- **Customizable Settings**: Settings panel for module visibility and goals

### Installation

1. Download the plugin files
2. Copy `main.js`, `styles.css`, and `manifest.json` to `.obsidian/plugins/smartwrite-companion/`
3. Restart Obsidian and enable the plugin in settings

### Usage

After enabling the plugin, a new sidebar view "SmartWrite Companion" will be available. Open it to see real-time metrics while editing your documents.

### Development

For local development:
```bash
npm install
npm run build
```

## Deutsch

SmartWrite Companion ist ein Plugin für Obsidian, das Echtzeit-Schreibanalysen bietet, um Ihre Produktivität zu verbessern und Ihren Fortschritt zu verfolgen.

### Hauptfunktionen

- **Echtzeit-Analyse**: Zählt Wörter, Absätze, Sätze und Silben sofort, während Sie schreiben
- **Lesemetriken**: Berechnet geschätzte Lesezeit und Lesegeschwindigkeit (WPM)
- **Sitzungsverfolgung**: Überwacht die Schreibzeit der aktuellen Sitzung und den täglichen Fortschritt
- **Tägliche Ziele**: Legen Sie tägliche Wortzahlziele fest und verfolgen Sie sie
- **Modulare Benutzeroberfläche**: Akkordeon-UI mit erweiterbaren Abschnitten für Metriken, Sitzung und Einstellungen
- **Leistungsoptimierung**: Intelligentes Cachesystem für sofortige Updates
- **Anpassbare Einstellungen**: Einstellungspanel für Modul-Sichtbarkeit und Ziele

### Installation

1. Laden Sie die Plugin-Dateien herunter
2. Kopieren Sie `main.js`, `styles.css` und `manifest.json` nach `.obsidian/plugins/smartwrite-companion/`
3. Starten Sie Obsidian neu und aktivieren Sie das Plugin in den Einstellungen

### Verwendung

Nach der Aktivierung des Plugins ist eine neue Seitenleiste "SmartWrite Companion" verfügbar. Öffnen Sie sie, um Echtzeit-Metriken beim Bearbeiten Ihrer Dokumente zu sehen.
