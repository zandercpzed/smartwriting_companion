## Português

SmartWrite Companion é um plugin para Obsidian que incorpora diversas ferramentas para enriquecer a escrita dentro do Obsidian.

### Objetivos

- Oferecer utilitários de escrita (snippets, templates, geração de texto)
- Integrações com IA e ferramentas externas (futuro)
- Melhorar fluxo de criação de conteúdo no Obsidian

### Desenvolvimento

Instale dependências e rode em modo de desenvolvimento conforme o template padrão de plugins do Obsidian.
