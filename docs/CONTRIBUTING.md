# Contributing to SmartWrite Companion

Obrigado por considerar contribuir!

- Fork e clone este repositório.
- Crie uma branch com o seu trabalho: `git checkout -b feat/minha-coisa`.
- Faça mudanças pequenas e commits claros.
- Abra um Pull Request descrevendo as mudanças.

Testes locais:

- Instale dependências: `npm install`
- Rode build: `npm run build`
- Teste no Obsidian copiando a pasta do plugin para o vault e ativando-o.

Padrões de código e estilo: adicione lint/format conforme necessário.
